===============================================================================
Indirect dependencies — Qt runtime libraries
===============================================================================

The Qt 6 distribution dynamically links against a set of permissively licensed
support libraries that ship alongside minNotes. These are covered by their own
upstream licenses (BSD / MIT / FTL / public domain / similar) and are
redistributed verbatim from the Qt build. Notable libraries include:

  * zlib (zlib license)
  * libpng (libpng / zlib-style)
  * libjpeg-turbo (BSD + IJG)
  * libtiff (libtiff / MIT-style)
  * libwebp + libsharpyuv (BSD)
  * brotli (MIT)
  * liblzma / xz (public domain / 0BSD)
  * FreeType (FTL or GPL-2)
  * HarfBuzz (MIT)
  * ICU (Unicode license)
  * PCRE2 (BSD)

For the canonical license texts of these libraries refer to the Qt 6 source
distribution at https://download.qt.io/official_releases/qt/. None of these
libraries impose copyleft obligations on minNotes.
