# Keyboard shortcuts

macOS uses **⌘** (Command); on Windows use **Ctrl** in its place. **⇧** is Shift.

## Documents

| Action | Shortcut |
|---|---|
| New note | ⌘N |
| Open… | ⌘O |
| Save | ⌘S |
| Save As… | ⌘⇧S |
| Close tab | ⌘W |

## Editing

| Action | Shortcut |
|---|---|
| Undo | ⌘Z |
| Redo | ⌘⇧Z &nbsp;or&nbsp; ⌘Y |
| Cut / Copy / Paste | ⌘X / ⌘C / ⌘V |
| Indent / outdent list item | Tab / ⇧Tab |

## Inline formatting

| Action | Shortcut |
|---|---|
| Bold | ⌘B |
| Italic | ⌘I |
| Underline | ⌘U |
| Strikethrough | ⌘⇧X |
| Inline code | ⌘E |
| Link | ⌘K |
| Highlight | ⌘⇧H |
| Clear formatting | ⌘\ |

## Tables

| Action | Shortcut |
|---|---|
| Fill selection down | ⌘D |
| Fill selection right | ⌘R |
| Select all cells | ⌘A |

## Video (while a video is active)

| Action | Shortcut |
|---|---|
| Rewind (hold) | A or J |
| Fast-forward (hold) | D or L |
| Cycle review speed (0.5×–2×) | R |
| Reset speed to 1× | ⇧R |

## In the document (markdown-style)

Typed at the start of a block — see [Editing](editing.md):

| Type | Becomes |
|---|---|
| `# ` … `###### ` | Heading 1–6 |
| `> ` | Quote |
| `- ` / `* ` / `+ ` | Bullet list |
| `1. ` | Numbered list |
| `- [ ] ` / `- [/] ` / `- [x] ` | Task: to-do / doing / done |
| ```` ``` ```` + `Enter` | Code block |
| `---` + `Enter` | Divider |
